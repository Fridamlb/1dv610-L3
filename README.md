# Textanalys - Document Analyzer

En webbappplikation för att analysera texter. Appen räknar ord, meningar, stycken, identifierar språk och mycket mer.

# Krav

| Krav | Beskrivning |
|----------|----------|
| Krav 1 | Användaren kan mata in text i ett textfält |
| Krav 2 | Användaren kan anända en knapp för att analysera texten |
| Krav 3 | Appen visar antal ord i texten |
| Krav 4 | Appen visar antal meningar i texten |
| Krav 5 | Appen visar antal stycken i texten |
| Krav 6 | Appen visar längsta ordet i texten |
| Krav 7 | Appen visar kortaste ordet i texten |
| Krav 8 | Appen analyserar och visar vilket språk texten är skriven i + procent av säkerhet |
| Krav 9 | appen visar Titlar/rubliker från texten |

# Information

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Modul: DocumentStats (utvecklad själv)
* Deployment: Railway

# Hur använder man appen? 

1. Gå till 
2. Skriv eller klistra in din text i textfältet
3. Klicka på "Analysera" knappen
4. Resultatet visar under knappen
